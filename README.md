# Oanh Tran Personal Website

This is a Jekyll/GitHub Pages personal academic website.

## Quick Map

```text
_pages/about.md              Main About page: bio, pronunciation, News, Publications
_pages/year-archive.html     Blog page at /blog/
_posts/                      Blog posts written as Markdown
_publications/               Publication entries written as Markdown
_config.yml                  Sidebar profile, social links, site settings
_data/navigation.yml         Top navigation links
_sass/                       Visual styling and color palette
_includes/publication-list.html  Publication list template
files/                       PDFs and downloadable files
images/                      Profile images and favicon assets
```

## Update About

Edit:

```text
_pages/about.md
```

This controls the visible About page, including:

- intro bio
- pronunciation block
- News
- Publications include

## Update Sidebar Profile

Edit the `author:` section in:

```text
_config.yml
```

Useful fields:

```yaml
avatar: "profile_avatar.jpeg"
name: "Oanh N. Tran"
pronouns: "she/her"
bio: "It is not hard; it is just new."
email: "..."
github: "..."
googlescholar: "..."
orcid: "..."
```

The sidebar quote styling is in:

```text
_sass/_sidebar.scss
```

## Update News

Edit the `News` list in:

```text
_pages/about.md
```

Add new items near the top:

```html
<li>
  <span class="news-date">MM.YYYY</span>
  <span class="news-text">Your news update here.</span>
</li>
```

## Add Publication

Create a file in:

```text
_publications/
```

Use a filename like:

```text
YYYY-MM-short-title.md
```

Template:

```markdown
---
title: "Paper title"
collection: publications
permalink: /publication/short-title
excerpt: "Short summary."
date: YYYY-MM-DD
venue: "Venue name"
paperurl: "https://paper-link"
arxiv: "https://arxiv-link"
authors: "<strong>Oanh N. Tran</strong>, Coauthor"
citation: "Citation text."
---

Longer paper description.
```

Publication list layout is controlled by:

```text
_includes/publication-list.html
_sass/_page.scss
```

## Add Blog Post

Create a Markdown file in:

```text
_posts/
```

Filename format:

```text
YYYY-MM-DD-short-title.md
```

Template:

```markdown
---
title: "Post title"
date: YYYY-MM-DD
permalink: /blog/post-title/
tags:
  - research
  - evals
---

Write the blog post here.
```

The Blog page automatically creates category sections from `tags`.

Blog page files:

```text
_pages/year-archive.html
_sass/_archive.scss
```

## Update Pronunciation

Pronunciation is in:

```text
_pages/about.md
```

Look for:

```html
data-pronounce-text="Oanh Trần"
data-pronounce-lang="vi-VN"
```

The sound is generated by the browser with Web Speech API. No audio file is stored.

## Update Profile Images

Replace:

```text
images/profile_image.jpeg
images/profile_avatar.jpeg
```

`profile_avatar.jpeg` is the sidebar image. The sidebar points to it from `_config.yml`.

## Update Resume

Replace:

```text
files/My_resume_2026_Jan.pdf
```

Then update any links in `_pages/about.md` if the filename changes.

## Update Colors

Palette lives in:

```text
_sass/_variables.scss
```

Current custom colors:

```scss
$main-blue: #224E87;
$accent-red: #C94C4C;
$site-background: #F5EFE6;
$support-blue: #7FB3E6;
$support-green: #9CAF88;
```

## Update Navigation

Edit:

```text
_data/navigation.yml
```

Current pages are About and Blog.

## Preview Locally

Install dependencies:

```bash
bundle install
```

Run:

```bash
bundle exec jekyll serve
```

Open:

```text
http://localhost:4000
```

Note: this project may require Ruby 3+ and the Bundler version in `Gemfile.lock`.

## Do Not Edit

Usually avoid editing:

```text
_layouts/
_includes/seo.html
assets/js/main.min.js
assets/fonts/
assets/webfonts/
```

`_site/` is generated build output. It can be deleted and should not be edited manually.
