# Blog posts

Write blog posts as Markdown files in this `_posts` folder.

Use this filename format:

```text
YYYY-MM-DD-short-title.md
```

Start each post with front matter like this:

```yaml
---
title: "Post title"
date: 2026-06-28
permalink: /blog/post-title/
tags:
  - research
---
```

Everything below the front matter is normal Markdown.
